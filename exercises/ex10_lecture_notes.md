# 經濟學入門 — 第三週白板筆記

## 主題：供給與需求

### 需求曲線
- 價格↑ → 需求量↓（負斜率）
- 位移因素：所得變動、偏好改變、替代品價格
- 例子：珍珠奶茶漲價 → 買的人變少 → 但如果大家突然都愛喝 → 整條曲線右移

### 供給曲線
- 價格↑ → 供給量↑（正斜率）
- 位移因素：成本、技術、預期
- 例子：珍珠原料漲價 → 供給減少 → 曲線左移

### 均衡
- 供需交叉點 = 均衡價格
- 超額供給 → 價格下降
- 超額需求 → 價格上升
