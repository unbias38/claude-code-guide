# 補助提案範本：偏鄉數位教育計畫

## 提案摘要
本計畫旨在為台灣 10 所偏鄉國小導入數位學習設備與課程，
預計 12 個月內服務 500 名學生。

## 預算摘要
| 項目 | 金額 |
|------|------|
| 平板電腦（100台）| NT$ 1,500,000 |
| 網路設備 | NT$ 300,000 |
| 師資培訓 | NT$ 400,000 |
| 課程開發 | NT$ 500,000 |
| 行政管理 | NT$ 300,000 |
| **合計** | **NT$ 3,000,000** |

## 過去成果
- 2024年服務 3 所學校、150 名學生
- 學生數學成績平均提升 18%
- 家長滿意度 92%
