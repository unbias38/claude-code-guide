# 實作練習素材

> 這些檔案是「Claude Code 學習指南」各章節延伸實作練習的模擬素材。
> 同學可以直接用這些檔案搭配 Claude Code 做練習。

## 使用方式

1. 找到你要做的練習（例如「實作 14：合約審查標記與談判」）
2. 把對應的素材檔案放到你的練習資料夾
3. 啟動 Claude Code，按照練習指示操作
