# 微積分筆記 — 定積分

## 基本觀念
定積分 = 曲線下面積
∫[a到b] f(x)dx = F(b) - F(a)

## 我的筆記（手寫轉打字）
- 基本公式：∫x^n dx = x^(n+1)/(n+1) + C
- 記住：n≠-1 的時候才能用！
- ∫1/x dx = ln|x| + C
- 常忘記：∫e^x dx = e^x + C（不是 e^x/x！）
