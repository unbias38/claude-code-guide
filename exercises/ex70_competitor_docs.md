# 競品分析素材

## 競品A：QuickBooks（Intuit）
- 定價：US$30-200/月
- 優勢：品牌知名度高、功能最完整、大量整合
- 劣勢：不支援繁中、台灣發票格式不支援、學習曲線陡
- 評價：G2 評分 4.0/5（3,200 則）

## 競品B：波奇記帳（台灣本土）
- 定價：免費 / Pro NT$199/月
- 優勢：支援台灣發票、中文介面、價格便宜
- 劣勢：報表功能陽春、沒有庫存管理、客服慢
- 評價：App Store 4.3/5（580 則）

## 競品C：Xero
- 定價：US$15-78/月
- 優勢：介面漂亮、API 開放、多幣別支援
- 劣勢：繁中翻譯不完整、台灣在地化不足
- 評價：G2 評分 4.3/5（2,100 則）

## 我們的產品定位
- 目標：台灣中小企業
- 差異化：AI 自動分類 + 完整台灣發票整合 + 中文客服
- 定價策略：NT$299-799/月（比 QuickBooks 便宜，比波奇完整）
