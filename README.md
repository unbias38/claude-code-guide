# Claude Code 完全學習指南：從入門到精通

從零開始學 Claude Code 的繁體中文教學指南，適合完全不會寫程式的人。

> **聲明：** 本指南為社群自發整理的學習資源，非 Anthropic 官方出版物。Claude 和 Claude Code 是 [Anthropic](https://www.anthropic.com/) 的產品。85 個實作練習改編自 [Claude 官方使用案例](https://claude.com/resources/use-cases/)，每個練習附有原始連結。

## 線上閱讀

👉 **[打開講義](https://unbias38.github.io/claude-code-guide/)**

## 內容

- **34 章教學** — 從「終端機是什麼」到「多 Agent 並行開發 + 部署上線」
- **85 個實作練習** — 涵蓋非營利、商務、財務、法律、科學、個人生活等場景
- **81 個練習素材** — 模擬的 CSV、合約、CRM 資料等，不用自己準備
- **貫穿專案「我的記帳本」** — 從一個 HTML 小工具一路升級成全端 App

## 下載練習素材

練習需要用到 `exercises/` 資料夾裡的素材檔案。兩種方式取得：

**方式 1：下載 ZIP**

點頁面上方綠色的 **Code** → **Download ZIP**，解壓縮即可。

**方式 2：Git Clone**

```bash
git clone https://github.com/unbias38/claude-code-guide.git
```
